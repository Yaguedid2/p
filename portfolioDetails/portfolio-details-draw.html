LaLinea AI Drawing Game

We have developed a game using Unity and AI-based image recognition. The game is based on the old cartoon named LaLinea, and it is not only a fun drawing game, but it is also an educational tool for little kids to learn how to draw. It contains various mini-games that aim to help kids learn different skills while they play.

The game is able to recognize doodles with an accuracy of over 95% thanks to the use of a powerful AI model that has been trained on a data set of 1.5 million images. The data for this model was collected using the popular Google application QuickDraw, which allows users to draw images that are then used to train the AI model. This allowed us to create a diverse set of 345 classes, which the game is able to recognize and respond to. The model is trained as a TensorFlow.js model and stored in a dedicated Node.js server, this allows us to use the model in the game and keep the game light and fast.

In addition to the drawing feature, the game also includes other educative mini-games such as:

  • An algorithm game: where kids can learn the basics of algorithms and problem-solving by completing a series of increasingly difficult puzzles.
  • A procedurally generated map game: where players have to remember the route towards the end and learn how to navigate through it.

The game also includes a scoring system, leaderboard, and various levels with different challenges. The game also has a leaderboard showing the level of happiness, anger, hunger and tiredness. Players can draw a car or a house to reduce tiredness, an apple to reduce hunger and so on. This creates a fun and interactive experience for kids as they can test their skills and see how accurately they can recognize their doodles and how they can improve the mood of the character in the game.
